> **The tool that has been delayed for nearly half a year is finally completed.**

## Project Overview

PowerZone is a web platform designed for gym-goers to explore available gym locations, trainer profiles, and membership plans. The platform features a responsive UI with intuitive navigation and is currently expanding to support user authentication.

![PowerZone Gym Website Subscription App](../../../public/assets/images/projects/powerzone-website.png)

## Role

UX/UI Lead – Collaborated in a multidisciplinary team.

## Key Features

- User registration/login and authentication (future enhancement).
- Ability to purchase training plans.
- Display of gym branches and trainer profiles.
- Subscription plan browsing.
- Planned features: user registration and login, JWT-based session handling.

## Value Delivered:

A modern, user-friendly app successfully deployed based on the web.

## Architecture Overview:

## Frontend:

- Design: Figma for Prototyping and Wireframes.
- Framework: React and Node JS for implementation.
- Styling: Tailwind CSS for a modern and responsive design

## Backend:

Framework: Express.js to handle server logic and APIs.
Database: PostgreSQL for data storage.
Authentication: JWT (JSON Web Tokens) for secure, stateless session management.
